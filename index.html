Federal Trade Commission

The easiest way for users to navigate to accomplish
their goals on the website

Duration

October-November 2022

Role

UX/UI designer

Team

Della Figueroa, Galle Figaro, Yulee Lee

Tools

Figma, Google Drive, Zoom, InVision, Mockup

Project Overview

I found the fact that is design trends for government sites have very little to do with colors, fonts, or any sort of fleeting fashion. What matters most for government sites is trust, transparency, and ease of use. That’s why I focused on those 3 features to redesign this government agency (FTC) website.



Problem

It is difficult to navigate the government website. I wanted to set FTC apart by redesigning the site and making users want to visit the website and do things via the internet then coming in personally and spending lots of time in an office. The concept is to get important information for the users onto the site and make that a top priority.


Solution

Develop a rebrand of the FTC website so that we are able to find how to make the site the most accessible and ease of use for users.



User Statement

Rebecca’s identity was stolen so now she is using the FTC as her source of information on what steps to take next.

User Testing

The goal for this guerilla test was to ensure the user can find the contacts and their answers effectively. I am conducting this research to see if this website will be user-friendly and accessible.

UI Pain Points (Heuristic Issues)

After user tests, I found some heuristic issues from FTC website.
  • 1. Aesthetic and minimalist design - Get ride of 2 of “Report Fraud” buttons
  • 2. Help and documentation issues - Need live chatbot. It will be easy for users to search and focus on the their tasks.
  • 3. Match between system and the real world issues - Ensure users can understand the meaning without having to go look up a word’s definition
  • 4. Many unnecessary elements on the sub-navigation bar - Get rid of all the unnecessary elements



Redesign Process

The first thing I did was card sorting and sitemap for the better navigation. After creating the sitemap, I got a feedback from UX designer, Michelle.

“This sitemap looks great - however, I am concerned about how you might display the tertiary navigation especially since there's so much about it. Redesign iterations

This section highlights the redesign iterations after five-second usability test.


The "home" page

In order to keep users on the FTC website, users need a better navigation to accomplish their goals such as reporting identity theft, finding contact information, or filing an antitrust complaints etc. For thses goals, the “home” page is the most important, so I primarily focused on redesigning homepage.





Style Guide

I designed various symbols that were reused throughout the site, such as typography, colors, icons and buttons. I converted these symbols into components, and combined various components into the cards see around the prototype.


Typography

I chose only one style, Public Sans, which is a strong, and neutral typeface for display, headings, body, caption and footer. Government sites also tend to have public sans as a standard font to display the information, which creates uniformity throughout the website and makes the information easily accessible.

Color Palette

As I mentioned, users expect the trust, and transparency from the government. So I chose the trust color which is blue as a primary color.

Buttons

I used primary, secondary, and accent colors for buttons. The primary color is for default, the secondary one is for hover, and the accent is for active button. I even created all different sizes which are small, medium, and large. Then I can use these buttons for all different devices.



Usability Test

Midway through this project, I conducted another usability testing on seven potential users through Zoom. First I tested FTC’s navigation and then finding the contact information that the users need.

I was very intrigued to see that the pain points and features the testers requested were fulfilled in my prototype. It was interesting how important the contact information for users.


“I think the homepage looks great and I love the interaction nav. I also love the hovering cards. For the contact, you need to simplify, if you want it to. Even the text, the users would take time to read it. It would be better more straight for. I would also like the font in the contact menu a little bit bigger for accessibility”

-Della, Desktop tester

“It’s a bit confused to find the contact information but the government website has so many menus. It is hard to navigate no matter what.”

-Anonymous, Mobile tester



Final Design

The purpose of this project was to allow users to more quickly and easily navigate to accomplish their goals. With this in mind, I wanted the FTC website to have cleaner, simpler, and easier to use.

After the last usability test, I iterated all features that testers pointed out such as bigger font, and easier way to find contact information that the users need.

Takeaways

Onboarding tutorial for navigation

Users were unsure which menu they need to go to. It would greatly benefit from an onboarding tutorial in order to understand the navigation better.

Redesign other features

Since this project strictly focused on the homepage for the better navigation, the next step would be to redesign other pages.

All in one website

I realize that FTC has multiple websites to accomplish goals for users. It might be frustrating and confusing for users. It would be better all websites related to FTC in FTC website.
